# dotfiles

Config files for:
* bash
* git
* eslint
* tmux
* vim
* zsh

Thanks to Trevor D Miller for this [awesome set-up](https://trevordmiller.com/blog/dotfiles).

# Setup

- Clone this repo
- `cd` to the directory where you cloned it
- Run the set up script: `./index`
